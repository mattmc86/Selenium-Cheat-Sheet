SSL and Cookies

This is for disabling cookies and blocking pop ups and allowing you to access insecure certificates if you ever need to. You need to create a new object of the ChromeOptions class ChromeOptions options = new ChromeOptions(); before doing anything.
Block pop ups options.setExperimentalOption("excludeSwitches",
Arrays.asList("disable-popup-blocking"));
Set directory for file downloads
Map prefs = new HashMap();
prefs.put("download.default_directory", "/directory/path");
options.setExperimentalOption("prefs", prefs);

Accept insecure certificates options.setAcceptInsecureCerts(true);
Delete cookies driver.manage().deleteAllCookies();
Get screenshot and save on desktop
File src = ((TakesScreenshot)driver).getScreenshotAs(OutputType.FILE);
FileUtils.copyFile(src, new File ("/Users/matthewmckenna/Desktop/screenshotauto.png"));
