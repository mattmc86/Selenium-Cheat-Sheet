Getting Started

To create your automation project you first need to install the following:

  • Java - https://www.oracle.com/java/technologies/java-se-glance.html
  • ChromeDriver (or whatever browser(s) you plan on automating) - https://chromedriver.chromium.org/downloads
  • Eclipse or another IDE to write your code - https://www.eclipse.org/downloads/

You then have to set Java as your home path, open terminal and follow below steps.

  • vim ~/.bash_profile
  • export JAVA_HOME=$(/usr/libexec/java_home)
  • Hit Esc
  • type :wq then hit enter
  • source ~/.bash_profile
  • echo $JAVA_HOME

Then open Eclipse and set up a new workspace, create a new Java Project and add a new class and then you're ready to go!

After you have created your project and added your first class the below lines of code need to be added
System.setProperty("webdriver.chrome.driver", "/Users/matthewmckenna/Desktop/automation/chromedriver");
WebDriver driver = new ChromeDriver();
The first one is setting the location of where your chromedriver is saved on your system. The second one is creating an object of the ChromeDriver class you can access all of its methods.
