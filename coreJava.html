Loops

Loops allow you to peform an action(s) on a number of items.

If you take our above arrays we can iterate through the array and print out each item. Firstly set your iterator as 0, then set how many times it should iterate, in this case its the length of the array, then what to do after each iteration. IN this case we are increasing i by one. So from when i = 0 until i = the length of the array peform the action in the block. This is a simple action of printing the value of i in the array.

for(int i=0; i<myArray.length> i++)
{
System.out.println(myArray[i]);
}
You can also have enhanced loops like the below which would print out each item in array.
for(String s: arr3)
{
System.out.println(s);
}

Conditional Statements

Conditional statements will execute a code block if certain conditions are met.

If we take the above array and this time instead of printing out each item, we will only print it if the value is greater than 20.
for(int i=0; i i++)
{
if(myArray[i] > 20)
{
System.out.println(myArray[i]);
}
}
So in this case it will only print 30 and 40.
You can also have multiple conditions so this time we will print a message depending on the the value.
for(int i=0; i<myArray.length> i++)
{
if(myArray[i] > 20)
{
System.out.println(myArray[i]+" is higher than 20");
}
else
{
System.out.println(myArray[i]+" is equal or smaller than 20"); }
}
If you have many different conditions it is best to use a case statement rather than if.

Streams

Streams are a brilliant way to optimise your code. You basically, just chain methods onto each other.

If you think of a for loop with a conditional statement and how many lines of code there are. Then compare with using Java Streams. In the below example, we are going to create an array of names, then we want to only get names that ends with "n". Then we want to change each of those to upper case. Finally, we will print them in the console.
Stream.of("Ciaran","Matty","Boydy","Liam","Kieran").filter(s->s.endsWith("n")).map(s->s.toUpperCase()) .forEach(s->System.out.println(s));
This will print CIARAN KIERAN. If you wanted to put this into a new list then you could:
List<String>ls = Stream.of("Ciaran","Matty","Boydy","Liam","Kieran").filter(s->s.endsWith("n")).map(s->s.toUpperCase()) .collect(Collectors.toList()); ls.get(0);
so now you have a new list called "ls".

You can also remove duplicates and sort lists into an order. Create the list List <Integer>values= Arrays.asList(4,4,6,8,2,3,8,9,22,4,6,7,6); To do this simply values.stream().distinct().sorted().forEach(s->System.out.println(s));
You could also turn a array of strings which are numbers into an array of integers. For example,
String[] arr5 = {"1","2","4","5","6","75","40"};
int[] values2 = Arrays.stream(arr5).mapToInt(Integer::parseInt) .toArray();
