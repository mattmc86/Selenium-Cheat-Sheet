Properties


It is good practise to use a file to list data which would be reused in different tests. A good example is the browser. Under the main folder add a new folder called resources and then create new file in there. In that file you can set key value pairs (e.g. browser=chrome). Make sure you save the file with .properties extension so that you can use the Java Properties class to get methods.
Then when you create a new test in your method you use the Properties and FileInputStream to access that data from the file.
Properties prop = new Properties();
FileInputStream fis = new FileInputStream(System.getProperty("user.dir")+"//src//main//java//Automation//SeleniumFrameworkDesign//resources//GlobalData.properties");
prop.load(fis);
String browserName = prop.getProperty("browser");
